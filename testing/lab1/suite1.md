# Window Management

| ID | Description (описание) | Pre-Conditions (входные условия) | Actions\Steps (шаги) | Expected Results (ожидаемый результат) |
| :--- | :--- | :--- | :--- | :--- |
| **1.1** | Установка и первый запуск оконного менеджера niri | Установлена ОС Linux с доступом к Интернету и правами sudo. | 1. Выполните команды установки из README.md. 2. Выйдите из графической сессии. 3. Выберите сессию niri в дисплейном менеджере и войдите. | Сессия niri успешно запускается, на экране отображается чистый рабочий стол с обоями по умолчанию, графические сбои отсутствуют. |
| **1.2** | Открытие первого окна терминала в пустой сессии | Успешно выполнен кейс 1.1, на экране пустой рабочий стол. | Нажмите комбинацию клавиш Super+T для запуска терминала по умолчанию. | Окно терминала успешно открывается и отображается в виде вертикальной колонки на экране слева, занимающей всю доступную высоту. |
| **1.3** | Добавление второго окна в ленту | Успешно выполнен кейс 1.1, открыто ровно одно окно в фокусе. | Нажмите комбинацию клавиш Super+T для запуска терминала №2. | Новое окно терминала №2 открывается справа от первого окна в виде новой вертикальной колонки, фокус перемещается на терминал №2. |
| **1.4** | Перемещение фокуса между соседними окнами влево | Успешно выполнен кейс 1.1, открыто два окна (терминал №1 слева, терминал №2 справа), фокус ввода на правом окне (терминал №2). | Нажмите клавиатурное сочетание Super+Left (или Super+H). | Фокус ввода успешно перемещается на левое окно (терминал №1). |
| **1.5** | Прокрутка ленты окон за пределы экрана при вызове новых приложений | Успешно выполнен кейс 1.1, экран полностью заполнен тремя открытыми окнами, фокус на крайнем правом окне. | Нажмите Super+T для открытия четвертого окна терминала. | Лента окон плавно сдвигается влево, скрывая первое окно за левую границу экрана, а новое четвертое окно появляется справа в фокусе. |
| **1.6** | Интерактивное перемещение активного окна (колонки) вправо | Успешно выполнен кейс 1.1, открыто три окна (A, B, C по порядку слева направо), фокус ввода на среднем окне B. | Нажмите комбинацию клавиш Super+Shift+Right (или Super+Shift+L). | Активная колонка B перемещается на одну позицию вправо, меняясь местами с колонкой C (новый порядок колонок: A, C, B). |
| **1.7** | Изменение ширины выбранной колонки клавиатурой | Успешно выполнен кейс 1.1, открыто одно окно терминала в фокусе с шириной по умолчанию. | 1. Нажмите Super+R для изменения размера. | Ширина активной колонки увеличивается по заданным шагам (1/3, 2/3, 3/3), сдвигая при наличии соседние окна. |
| **1.8** | Закрытие активного окна в заполненной ленте | Успешно выполнен кейс 1.1, открыто три окна в ленте, фокус на среднем окне, левое окно частично скрыто за экраном. | Нажмите комбинацию клавиш Super+Q для закрытия текущего окна. | Активное окно закрывается, фокус переходит на соседнее правое окно, лента не перемещается. |
